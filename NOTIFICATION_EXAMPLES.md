# Notification Quick Reference

## What Each User Will See

### 📋 ANNOTATORS
```
When assigned a task:
🎯 John Smith assigned "Plastic Waste Dataset #1" to you for annotation
→ ACTION: Open Dashboard → Click image → Start annotating

When task is reviewed:
✅ Sarah Johnson approved "Plastic Waste Dataset #1" | ACTION: Check your Dashboard for next steps
OR
❌ Sarah Johnson rejected "Plastic Waste Dataset #1" | ACTION: Review feedback and resubmit if needed

When Melbourne user makes final decision:
✅ Mike Brown approved "Plastic Waste Dataset #1" for production | ACTION: Dataset is ready for use
OR
❌ Mike Brown rejected "Plastic Waste Dataset #1" | ACTION: Review feedback in Dashboard
```

### 🔍 TESTERS

## Key Features

✨ **User Names**: Always see who's doing the action
📸 **Image Names**: Always see which image it's about
➡️ **Clear Actions**: Instructions on exactly what to do next
🎨 **Visual Icons**: Quick visual identification of notification type
🔵 **Color Coding**: Green=Approved, Red=Rejected, Cyan=Action needed
⏱️ **Timestamps**: See when the notification was created

---

## How Notifications Appear

All notifications show:
1. **Icon** - Visual indicator of notification type
2. **Main Message** - WHO, WHAT, and WHICH IMAGE
3. **Action Instructions** - Next steps in blue highlighted box
4. **Timestamp** - Date and time of notification
5. **Mark as Read** - Check button to mark read (appears on unread only)

---

## Unread vs Read

- **Unread**: Light blue background, bold text, check button visible
- **Read**: Normal background, regular text, no check button

Click the bell icon to see all notifications, or mark as read one by one.
